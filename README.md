# Programming Task 03 - Loops, Patterns & Basic Automation

## Overview

This repository contains solutions for **Task 03: Loops, Patterns & Basic Automation** implemented in both **Python** and **C Programming**.

The purpose of this task is to strengthen understanding of:

* Loops and Iteration
* Pattern Printing
* Conditional Statements
* String Manipulation
* Basic Authentication Logic
* Problem Solving
* Basic Automation Concepts

# Python Programs

## 1. Multiplication Table Generator

### Description

Accepts a number from the user and displays its multiplication table from 1 to 10 using a for loop.

### Concepts Used

* User Input
* for Loop
* Arithmetic Operations

### Output

![Multiplication Table](Screenshots_Programming_03/Python_Screenshots/multiplication_table_py.png)

## 2. Number Analysis Tool

### Description

Accepts a number N and calculates:

* Sum of numbers from 1 to N
* Total even numbers
* Total odd numbers

### Concepts Used

* Loops
* Conditional Statements
* Counters

### Output

![Number Analysis](Screenshots_Programming_03/Python_Screenshots/number_analysis_py.png)

## 3. Pattern Printing Challenge

### Description

Prints different star and number patterns using nested loops.

### Concepts Used

* Nested Loops
* Pattern Problems
* Iteration

### Output

![Pattern Printing](Screenshots_Programming_03/Python_Screenshots/patterns_py.png)

## 4. Password Attempt Simulator

### Description

Simulates a simple login system with a predefined password and maximum three login attempts.

### Concepts Used

* while Loop
* Conditional Statements
* Authentication Logic

### Output

![Password Attempt Simulator](Screenshots_Programming_03/Python_Screenshots/password_attempt_py.png)
